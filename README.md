# Project-ocuhome2016
A Direct rip of the original oculus home from 2016 (specifically version 1.20.0.466754). Its basically a unity game, so i just ripped it and restored all the scripts. Fully open source and can be ported to all quest devices (except for the quest 1) but the PCVR version can be used on any vr headset that can run Revive and/or Steam VR.
